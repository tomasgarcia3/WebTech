Quiz 4-2: Value of $1000 at 5% (or any input) interest rate in 10 years

Use JavaScript to implement the following features:

  • User will enter one number for "Interest Rate". Clicking the button will start an animation that changes the value of "Year" and "Money" Every second. For example, after the first second, "Year" should change to 2023 and "Money" should change to 1050.00 (assuming interest rate is 5%.)
  • Every second, "Year" should increase by 1 and "Money" should increase based on the interest rate.
  • After year 2032, "Year" should be reset to 2022 and "Money" should be reset to 1000.
  • Please always keep two decimal places. Hint: toFixed.
